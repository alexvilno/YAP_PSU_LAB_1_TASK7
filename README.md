# Базовый класс Point
## Поля
Класс содержит поля:
```c#
private int x; //координата по оси х
private int y; //координата по оси y
private int z; //координата по оси z
```

## Конструкторы
Конструктор **по умолчанию** инициализирует точку в начале системы координат:

```c#
public Point() //конструктор по умолчанию
{
  x = 0;
  y = 0;
  z = 0;
}
```

Конструктор **от трех целочисленных параметров** инициализирует точку с координатами (x,y,z) соответственно:

```c#
public Point(int x, int y, int z) //конструктор от трех целочисленных параметров
  {
  this.x = x;
  this.y = y;
  this.z = z;
}
```

Кроме того, реализован **конструктор копирования**:

```c#
public Point(Point p) //конструктор копирования
{
  x = p.x;
  y = p.y;
  z = p.z;
}
```

# Дочерний класс Triangle

## Поля
Класс содержит поля:
```c#
private Point a; //вершина А
private Point b; //вершина B
private Point c; //вершина С
```
## Конструкторы
Конструктор **по умолчанию** инициализирует правильный треугольник:

```c#
public Triangle() //конструктор по умолчанию
{
  a = new Point(1, 0, 0);
  b = new Point(0, 1, 0);
  с = new Point(0, 0, 1);
}
```

Конструктор **от трех точек** (объектов класса **Point**) инициализирует треугольник с вершинами, совпадающими с этими точками. Принимает три параметра на вход &mdash; три объекта класса **Point**
```c#
public Triangle(Point a, Point b, Point c) //конструктор от трех точек(вершин)
{
  this.a = a;
  this.b = b;
  this.c = c;
}
```

Конструктор **копирования** выглядит так:
```c#
public Triangle(Triangle prev) //конструктор копирования
{
  this.a = prev.a;
  this.b = prev.b;
  this.c = prev.c;
}
```
